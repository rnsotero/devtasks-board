## 📌 Sobre o projeto

O **DevTasks Board** é um sistema de gerenciamento de tarefas inspirado no modelo **Kanban**, permitindo organizar atividades em diferentes etapas do fluxo de trabalho.

A aplicação foi desenvolvida com foco em **prática de JavaScript puro (Vanilla JS)**, manipulação do DOM e construção de uma interface interativa com uma boa experiência do usuário.

---

## 🚀 Funcionalidades

* ➕ Criar tarefas
* ✏️ Editar tarefas
* 🗑️ Remover tarefas
* 🔄 Drag and Drop entre colunas
* 💾 Persistência de dados com localStorage
* 🧠 Organização em colunas:

  * To Do
  * Em Progresso
  * Concluído
* 🪟 Modal para criação de tarefas (UX moderna)
* ⚡ Interface fluida com animações

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Web APIs (DOM + localStorage)

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

* Praticar **manipulação de DOM**
* Trabalhar com **eventos em JavaScript**
* Implementar **persistência de dados no navegador**

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/rnsotero/devtasks-board.git
```

2. Acesse a pasta:

```bash
cd devtasks-board
```

3. Abra o arquivo `index.html` no navegador
